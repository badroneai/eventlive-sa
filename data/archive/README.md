# Archived Releases

Each archived release lives under:

- `data/archive/releases/<release-id>/program.json`
- `data/archive/releases/<release-id>/release-notes.md`

Archive entries are indexed in:

- `reports/releases/archive-index.json`

Release-specific visitor-facing snapshots remain in:

- `reports/releases/<release-id>.program.html`
- `reports/releases/<release-id>.print.html`
- `reports/releases/<release-id>.share.html`
