# Preview Approval Checklist

- [ ] `program_title` matches the approved event name.
- [ ] `organizer_name` matches the approved organizer name.
- [ ] `venue` and `city` are correct.
- [ ] `event_start` and `event_end` reflect the approved event window.
- [ ] `updated_at` reflects the latest approved revision.
- [ ] Session ordering is correct.
- [ ] Session titles are final and typo-free.
- [ ] Speaker and moderator names are final and typo-free.
- [ ] Tracks and room assignments are correct.
- [ ] The preview in `dist/index.html` matches the approved program.
- [ ] Validation report is PASS before approval.
